메타데이터(Metadata)

메타데이터(Metadata)란 데이터에 대한 구조화된 데이터를 말합니다.

다시 말해, 특정 데이터 자원을 설명하고 식별하는 데이터입니다.

메타데이터의 종류

  1. 설명 메타데이터
  2. 구조 메타데이터: 데이터의 유형, 크기, 포맷, 필드 등에 관한 정보를 제공합니다.
  3. 관리 메타데이터: 데이터 접근권한, 암호화키, 저작권 등 데이터 관리와 관련된 정보입니다.
  4. 참조 메타데이터: 데이터의 출처, 버전, 관련 자원에 대한 정보 등이 포함됩니다.

이렇게 메타데이터는 데이터의 식별, 검색, 이용, 관리를 용이하게 해주는 중요한 역할을 합니다.
도서관의 서지정보,디지털 사진의 EXIF 정보, 웹페이지의 HTML 태그 등이 메타데이터의 예시입니다.
